# Bug Hunter (Python / Pygame)

Bug Hunter is a small arcade-style shooter written in Python using the Pygame library.

The project was created to explore gameplay programming concepts such as:

- Player movement and input handling
- Bullet firing systems
- Enemy spawning and difficulty scaling
- Collision detection
- Score and lives tracking
- Game state management (title screen, gameplay, game over)

## Controls

Arrow keys – Move left/right  
Space – Shoot  
S – Start game  
R – Restart after game over

## Technologies

Python  
Pygame

## Installation

Install pygame:

```
pip install pygame
```

Run the game:

```
python main.py
```
